# Blogger SEO Tools

Free SEO tools for Blogger users.

## Features

- Meta Tag Generator
- Open Graph Generator (planned)
- Twitter Card Generator (planned)
- JSON-LD Schema Generator (planned)
- Robots.txt Generator (planned)

## Installation

Clone repository:

```bash
git clone https://github.com/USERNAME/blogger-seo-tools.git
```

Open index.html in browser.
